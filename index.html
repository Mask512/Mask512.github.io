3 Hal dasar yang dipelajari di kelas ini

  • HTML

    HTML adalah salah satu markup language yang digunakan untuk membuat struktur dan menampilkan konten pada World Wide Web (Website)

  • CSS

    CSS adalah singkatan dari Cascading Style Sheets. Cascading Style Sheets adalah kumpulan perintah yang digunakan untuk menjelaskan tampilan sebuah halaman situs web dalam mark-up language. Mark-up language atau bahasa markah adalah bahasa pemrograman yang biasanya digunakan membuat website.

  • Javascript

    Javascipt adalah bahasa pemrograman komputer yang dinamis. Pada umumnya Javascipt digunakan pada web browser untuk menciptakan halaman web yang menarik, interaktif serta merapkan berbagai fungsi pada halaman web.
